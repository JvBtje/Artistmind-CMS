Make the plugin work?

1) Get a key at tinypng. https://tinypng.com/

2) Open index.php

3) In the top of index.php you find $tinypngkey = "";

4) put the key in the variable like this: $tinypngkey = "MYKEY";

5) Save it and now it should work.

6) Copy this plugin to the plugin cronjobs directory and outside this directory
